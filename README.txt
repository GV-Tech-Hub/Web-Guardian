WebVault: Site Access Control
============================

Description:
WebVault is a Chrome extension that provides PIN-protected access control for websites. It allows users to block access to specific domains and URLs, requiring a 4-digit PIN to access restricted content.

Purpose:
Created to help users:
- Maintain focus by blocking distracting websites
- Protect sensitive content from unauthorized access
- Implement parental controls
- Manage screen time and web habits

Features:
- 4-digit PIN protection
- Domain-level blocking
- Specific URL blocking
- URL whitelisting
- Dark mode interface
- Seamless navigation blocking
- Back button functionality

Requirements:
- Google Chrome Browser (Version 88 or higher)
- Chrome Extension Developer Mode enabled

Installation:
1. Download the extension files
2. Open Chrome and go to chrome://extensions/
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. Set up your initial 4-digit PIN when prompted

Usage Example:
A parent wants to restrict their child's access to social media during study hours:
1. Install WebVault
2. Set up a PIN
3. Add domains like "facebook.com", "instagram.com" to the blocklist
4. The child will need the PIN to access these sites
