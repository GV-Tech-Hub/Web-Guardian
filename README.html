WebVault: Site Access Control

🔒 Secure Website Access Management

WebVault is a powerful Chrome extension that provides PIN-protected access control for websites, helping users manage their web browsing habits and implement effective access restrictions.

✨ Key Features

  • 4-digit PIN protection system
  • Domain-level blocking
  • Specific URL blocking
  • URL whitelisting capabilities
  • Dark mode interface
  • Seamless navigation control

🚀 Getting Started

  1. Clone this repository
  2. Open Chrome Extensions (chrome://extensions/)
  3. Enable Developer Mode
  4. Click "Load unpacked"
  5. Select the extension directory

💡 Use Cases

  • Parental Controls
  • Productivity Management
  • Content Restriction
  • Screen Time Management
